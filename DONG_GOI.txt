===== HƯỚNG DẪN ĐÓNG GÓI VÀ PHÂN PHỐI FAZZYTOOL =====

Công cụ FAZZYTOOL đã được thiết kế để dễ dàng đóng gói và phân phối. Dưới đây là các bước để đóng gói và phân phối công cụ:

1. CẤU TRÚC THƯ MỤC CẦN THIẾT
   - Thư mục gốc (FazzyTool)
     - Các file Python (.py): main.py, browser_image.py, browser_video.py, gemini_prompt.py, prompt_loader.py, __init__.py
     - Các file batch (.bat): setup.bat, run.bat
     - File requirements.txt
     - File .env (sẽ được tạo khi chạy setup.bat)
     - Thư mục python (chứa Python embeddable)
     - Thư mục output (để lưu kết quả)

2. CHUẨN BỊ PYTHON EMBEDDABLE
   - Đảm bảo thư mục "python" chứa bản Python embeddable đầy đủ
   - File setup.bat sẽ tự động cấu hình Python embeddable và cài đặt các gói cần thiết

3. ĐÓNG GÓI
   - Nén toàn bộ thư mục FazzyTool thành file ZIP
   - Hoặc tạo file cài đặt sử dụng công cụ như NSIS, Inno Setup

4. HƯỚNG DẪN NGƯỜI DÙNG
   - Giải nén file ZIP (nếu đóng gói dạng ZIP)
   - Chạy file setup.bat để thiết lập ban đầu
   - Sau khi thiết lập, chạy file run.bat để sử dụng công cụ

5. YÊU CẦU HỆ THỐNG
   - Windows 10/11
   - Kết nối internet
   - Tài khoản Freepik Premium
   - API key Gemini (nếu sử dụng chế độ AI)

6. LƯU Ý KHI PHÂN PHỐI
   - KHÔNG bao gồm file .env có chứa thông tin cá nhân
   - Đảm bảo thư mục output trống
   - Đảm bảo đã xóa các file tạm và cache

===== KHẮC PHỤC SỰ CỐ =====

1. LỖI KHÔNG TÌM THẤY MODULE
   - Chạy lại file setup.bat để cài đặt lại các gói cần thiết
   - Kiểm tra file requirements.txt đã có đầy đủ các gói cần thiết chưa

2. LỖI TRÌNH DUYỆT
   - Chạy lại việc cài đặt trình duyệt: chọn tùy chọn 3 trong menu chính
   - Hoặc chạy lệnh: python\Scripts\playwright.exe install firefox

3. LỖI API KEY HOẶC COOKIE
   - Cập nhật lại thông tin trong file .env
   - Hoặc chạy lại setup.bat và chọn Y khi được hỏi về việc cập nhật file .env
