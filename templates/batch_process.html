Sử dụng CLI

Để sử dụng chức năng batch, mở Command Prompt/Terminal và chạy:

# Vào thư mục FazzyTool
cd "E:\Tool AI\FazzyTool"

# Tạo batch prompt AI
py main.py ai-batch --topics "con mèo" "con chó" "phong cảnh"

# Chạy batch processing
py main.py batch

# Chuyển ảnh thành video
py main.py images-to-videos
